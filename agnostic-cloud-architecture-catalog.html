Cloud Architecture, distilled.

Every cloud provider markets a different vocabulary for the same fundamental building blocks. This reference distills the architecture into fourteen components and sixty-five distinct types — and shows you the canonical equivalents in AWS, Azure, GCP, and the open-source ecosystem.
